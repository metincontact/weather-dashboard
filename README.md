# Weather Dashboard

A real-time weather app built with React, TypeScript, and Tailwind CSS.

## Features

- Search any city for current weather
- Temperature, feels like, humidity, wind speed
- 5-day forecast with interactive line chart
- Loading spinner and error handling
- Fully responsive dark theme

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Recharts
- OpenWeather API
- Vite

## Live Demo

[weather-dashboard-matin.vercel.app](https://weather-dashboard-matin.vercel.app)

## Getting Started

```bash
cp .env.example .env
# Add your OpenWeather API key to .env
npm install
npm run dev
```
